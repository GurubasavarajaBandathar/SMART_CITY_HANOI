Course Introduction

This course centers on the design and analysis of algorithms, emphasizing their real-world applications. The primary objective is to equip students with the skills to design algorithms tailored to specific tasks, implement them effectively using appropriate data structures, and optimize their efficiency. By bridging theoretical foundations with practical applications, this course provides essential tools for solving a wide range of computational problems.

DS/Algorithms Subcategory Topics
Linear Structures Arrays 1-dimensional, 2-dimensional
Linked Lists Singly Linked List (SLL), Circular SLL, Doubly Linked List (DLL), Circular DLL
Non-linear Structures Trees n-ary Trees, Binary Trees, Binary Search Trees, AVL Trees, etc.
Graphs Weighted/Unweighted, Directed/Undirected
Others Heaps, HashMaps, Union-Find (Disjoint Set Union)
Algorithms Sorting Algorithms Bubble Sort, Merge Sort, Quick Sort, etc.
String Matching Algorithms Brute Force, Boyer-Moore, Knuth-Morris-Pratt (KMP), Rabin-Karp
Graph Algorithms BFS, DFS, Kruskal’s Algorithm, Dijkstra’s Algorithm, etc.
Backtracking Algorithms N-Queen Problem, Subset Generation

Real-World Application

Sorting and Searching: Used in e-commerce platforms and databases.

Graph Algorithms: Social networks, navigation systems.

Network Routing: Used in routing protocols for data transmission.

Finance: Real-time data analysis, predictive analytics, fraud detection.

Task Handling: Heaps used in priority queues.
